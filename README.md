<h1 align="center"><b>🚀 SIRADJU — Village Budget Planning Desktop Application</b></h1>

---

## 📌 Overview

**SIRADJU (Sistem Rancangan Anggaran Desa Junrejo)** is a desktop-based application designed to assist village governments in planning and calculating development budgets efficiently and transparently.

The application focuses on automating the calculation of work volumes and **Rencana Anggaran Biaya (RAB)** for public infrastructure projects. By digitizing the budgeting process, SIRADJU helps reduce manual errors, improve accuracy, and ensure structured financial reporting aligned with official standards.

Built as a standalone desktop system, SIRADJU is optimized for offline usage, making it suitable for environments with limited internet connectivity.

---

## 🎯 Key Highlights

- Desktop-based application optimized for offline use  
- Automated RAB (Budget Plan) calculation system  
- AHSP (Unit Price Analysis) processing via PDF upload  
- Structured financial reporting aligned with standards  
- Exportable reports in Excel format (RAB-ready template)  

---

## ✨ Features

- 📄 Upload & process AHSP data (PDF-based)  
- 📐 Work volume calculation system  
- 💰 Cost estimation & budgeting automation  
- 📊 Structured financial reporting  
- 📥 Export reports to Excel (formatted RAB template)
