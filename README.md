<h1 align="center"><b>🚀 SIRADJU — Village Budget Planning System</b></h1>

---

## 📌 Overview

**SIRADJU (Sistem Rancangan Anggaran Desa Junrejo)** is a web-based application designed to assist village governments in planning and calculating development budgets efficiently and transparently.

The system focuses on automating the calculation of work volumes and **Rencana Anggaran Biaya (RAB)** for public infrastructure projects. By digitizing the budgeting process, SIRADJU helps reduce manual errors, improve accuracy, and ensure structured financial reporting aligned with official standards.

---

## 🎯 Key Highlights

- Automated RAB (Budget Plan) calculation system  
- AHSP (Unit Price Analysis) data processing via PDF upload  
- Structured financial reporting system  
- Exportable reports in Excel format (RAB-ready template)  
- Designed for government-level administrative use  

---

## ✨ Features

- 📄 Upload & process AHSP data (PDF-based)  
- 📐 Work volume calculation system  
- 💰 Cost estimation & budgeting automation  
- 📊 Structured financial reporting  
- 📥 Export reports to Excel (formatted RAB template)
